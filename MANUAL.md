# Terminal Startpage – User Manual

## Commands
- config → show and edit settings
- ls -a → list all commands
- aliases → user-defined shortcuts

## Input Rules
- Command → execute
- Alias → open mapped site
- URL → open directly
- Otherwise → search DuckDuckGo

## Backup
- Settings stored in browser data
- Export JSON backup
- Import JSON to restore

## Warning
Clearing browser data deletes customization.
Always keep a backup.
